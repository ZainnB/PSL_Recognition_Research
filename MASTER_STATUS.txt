🆘 QUICK TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════════

Q: "Where do I start?"
A: Run this: conda activate fyp_env && python train.py

Q: "Tests fail"
A: Run: conda activate fyp_env && python test2.py
   This shows what's wrong with your environment.

Q: "CUDA out of memory"
A: Edit Config.BATCH_SIZE in train.py (reduce from 16 to 8)

Q: "Training is slow"
A: Check: GPU should be >90% utilized
   If low, there's a CPU bottleneck

Q: "Want to monitor progress"
A: Check: logs/history_*.json (updates each epoch)

Q: "Training interrupted"
A: Resume: python train.py --resume checkpoints/model_best.pt
